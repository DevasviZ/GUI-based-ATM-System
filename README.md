# ATM System

This is a simple ATM (Automated Teller Machine) system implemented in Java using Swing for the graphical user interface.

## Features

- Balance inquiry
- Withdrawal
- Loan management
- Deposit
- Receipt generation

## Getting Started

To run the ATM system on your local machine, follow these steps:

1. Clone the repository to your local machine:
   bash
   git clone https://github.com/your-username/atm-system.git
Open the project in your preferred Java IDE.

Compile and run the ATm_Sys.java file.

Usage
Upon running the application, you'll be presented with a GUI representing the ATM interface. You can perform the following actions:

Enter your PIN and press ENTER to access your account.
Use the buttons to navigate and perform various transactions.
Generate a receipt for loan transactions.
